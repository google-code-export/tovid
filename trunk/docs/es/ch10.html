1. La jerarquía de ficheros tovid
Nota

Este capítulo de la documentación contendrá a su debido tiempo una diversidad de ideas de diseño y otro material de interés primariamente para desarrolladores.

1. La jerarquía de ficheros tovid

He aquí la lista de todos los ficheros instalados por la versión 0.18 de tovid:

/usr/local/bin/
    dvrequant
    idvid
    makedvd
    makemenu
    makeslides
    makevcd
    makexml
    postproc
    tovid
    tovid-batch
    tovid-interactive
    tovid-test

Si instalas el paquete separado con el interfaz gráfico (tovid-gui), también tendrás:

/usr/local/bin/
    icons/
        tovid_icon_32.png
        tovid_icon_48.png
        tovid_icon_64.png
        tovid_icon_128.png
    tovidgui.py

Cuando ejecutes el script tovid script, obtendrás dos ficheros más en un directorio oculto en tu directorio hogar:

~/.tovid/
    stats
    tovid.config

Antes de de que pase mucho tiempo, tovid ncesitará muchos más ficheros que estos. Próximamente, el interfaz gráfico se dividirá en varios módulos Pyhon, y se añadirán algunos ficheros globales de configuración. La documentación HTML estará disponible como un paquete separado, e incluirá múltiples traducciones, y también los ficheros DocBook XML de partida usados para generar los HTML.

Aquí está la estructura dedirectorios planeada para alguna posible versión futura:

/usr/share/
    tovid/
        bin/
            dvrequant
            idvid
            makedvd
            makemenu
            makeslides
            makevcd
            makexml
            postproc
            tovid
            tovid-batch
            tovid-interactive
            tovid-test
            tovidgui.py
        
        config/
            tovid.config

        docs/
            source/
                tovid_documentation.en.xml
                tovid_documentation.fr.xml
                tohtml
            images/
                (all .jpg and .png illustrations)
            english/
                images/ -> /usr/share/tovid/docs/images/
                tovid_style.css -> /usr/share/tovid/docs/tovid_style.css
                index.html
                ch01.html
                ch02.html
                ..
                ..
            french/
                images/ -> /usr/share/tovid/docs/images/
                tovid_style.css -> /usr/share/tovid/docs/tovid_style.css
                index.html
                ch01.html
                ch02.html
                ..
                ..

            index.html (choose language)
            tovid_style.css

        gui/
            (wxPython GUI modules)

        icons/
            tovid_icon_32.png
            tovid_icon_48.png
            tovid_icon_64.png
            tovid_icon_128.png

        README
        GNU_GPL
        ChangeLog

/usr/bin/
    dvrequant           -> /usr/share/tovid/bin/dvrequant
    idvid               ..
    makedvd             ..
    makemenu            ..
    makeslides          ..
    makevcd             .. (symbolic links)
    makexml             ..
    postproc            ..
    tovid               ..
    tovid-batch         ..
    tovid-interactive   ..
    tovid-test          -> /usr/share/tovid/bin/tovid-test

~/.tovid/
    stats
    tovid.config
    tovidgui.prefs

Todos esto es extremadamente provisional, pero da una idea de qué hay en el futuro de tovid.
