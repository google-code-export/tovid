Classes

Element
A TDL element (Disc, Menu, or Video) with associated options.

    The Element is automatically filled with default values for the
    corresponding TDL element (from element_options); these may then
    be overridden by user-defined values.
OptionDef
Defines a single option, its default value, and documentation on its
    usage and purpose.
    
    An example:
        
        OptionDef(
            'debug',
            '[no|some|all]',
            'some',
            "Amount of debugging information to display"
           )

    This is a definition of a 'debug' option, along with a human-readable
    string showing expected argument formatting, a default value for the
    option, and a string documenting the purpose and usage of the option.
